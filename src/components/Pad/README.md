# Pad

`Pad/` contains native touch-first helpers that still belong in `orbmobile-ui`.

## Public API

- `MNumericKeypad`
- `usePadLayout`

## Intent

- Keep only native utilities that are still useful in this package
- Do not mirror the entire web pad surface from `orbcafe-ui`
- Keep demo-only pad screens out of the library

## Notes

- The numeric keypad remains a supported native component
- Layout helpers stay here for touch-first screen adaptation
